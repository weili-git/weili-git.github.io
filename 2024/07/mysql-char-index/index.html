MySQL Char & Index

July 22, 2024 in Web

1. Types of char

1.1 CHAR

  • fixed size: 1~255
  • padding with space
  • fast query speed

1.2 VARCHAR

  • dynamic size: 1~65535
  • extra byte to store length
  • slow query speed

1.3 TEXT

  • dynamic size: 1~65535
  • extra 2 bytes to store length
  • slowest query speed

2. Index

Create index for database can speed up query.

2.1 How to create index

create index idx_name
on employee(name);
-- or
alter table employee
add index idx_name(name);

-- unique index
create unique index idx_name
on employee(name);

2.2 How to check index

show index from employee;

2.3 How to drop index

drop index idx_name
on employee;
